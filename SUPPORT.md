# Support

## How to get help

This project uses [GitHub Issues](../../issues) to track bugs and feature requests.

**Before opening a new issue:**
1. Search existing issues to avoid duplicates
2. Check the [documentation](docs/) for answers
3. Review the [troubleshooting guide](docs/technical-guide.md#troubleshooting)

**For questions and discussions:**
- Use [GitHub Discussions](../../discussions) for general questions
- Tag your discussion with `question` or `help-wanted`

**For bugs:**
- Use the [Bug Report template](../../issues/new?template=bug_report.md)
- Include steps to reproduce, expected vs actual behavior, and your environment

**For feature requests:**
- Use the [Feature Request template](../../issues/new?template=feature_request.md)

## Microsoft Support Policy

Support for this project is limited to the resources listed above. This is an open-source project and is **not** covered by Microsoft Azure Support plans.
